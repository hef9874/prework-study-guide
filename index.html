Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements inside other elements are called child elements.
  • The script element embeds client script in the HTML document.
  • The base element defines the base URL for a website.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We will use the 'separation of concerns', which means that we should separate code into different files based on use.
  • The external css style sheet is the most commonly preferred css style sheet.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • You must always work from a branch off of the main branch and never on the main branch.
  • Git is a version control system that lets you manage each version of your project locally.
